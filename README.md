#draggable.js

High performance drag and drop in a tiny package

## Usage

**HTML**
```html

<div id="id"></div>

```

**JavaScript**

*Using browser globals:*
```js
var element = document.getElementById('id');
new Draggable (element);
```

*Using AMD/CommonJS:*
**JavaScript**
```js
var Draggable = require ('Draggable');
var element = document.getElementById('id');
new Draggable (element);
```
